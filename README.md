Sigmaplex is a web-based movie ticket booking system inspired by real cinema platforms.  
The system provides movie listings, news, and promotional information to enhance user experience.

## System Overview

Sigmaplex allows users to browse currently showing and upcoming movies, view movie details, and access cinema news and promotions through a user-friendly web interface.

## Features

- Browse currently showing and upcoming movies
- View movie details, news, and promotional campaigns
- User-friendly and responsive web interface
- Structured content management for movies, news, and promotions
